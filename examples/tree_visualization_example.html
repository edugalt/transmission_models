Tree Visualization Library - Interactive Demo

About This Demo

This demonstration shows how to use the Tree Visualization Library to display transmission trees from the transmission_models package. The library provides interactive visualizations with the following features:

  • Hierarchical Layout: Traditional tree layout based on parent-child relationships
  • Infection Time Layout: Nodes positioned by their infection time for temporal analysis
  • Interactive Highlighting: Hover over nodes to highlight related branches
  • Sampling Status: Visual distinction between sampled (blue) and unsampled (orange) hosts
  • Responsive Design: Automatically adapts to container size

Data Format

The library expects JSON data with the following structure:

{
    "log_likelihood": number,
    "tree": {
        "name": string,
        "index": number,
        "Infection time": number,
        "Sampled": boolean,
        "root": boolean,
        "children": [...]
    },
    "parameters": {...}
}
